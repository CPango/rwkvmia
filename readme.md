# rwkvmia

This project studies membership inference attacks for autoregressive language models.

## Main targets
- meta-llama/Llama-3.2-1B
- RWKV

## Main idea
- controlled fine-tuning setting
- G-Drift-style feature drift
- RWKV state drift extension

## Notes
- local machine is mainly for project setup and code drafting
- formal experiments will run on remote server with conda
